# Metrics

- Data contract pass rate: passed data checks divided by total data checks.
- Metric contract pass rate: passed metric contract checks divided by total metric checks.
- Metric variance percent: absolute certified vs uncertified variance divided by certified value.
- Trust score: weighted score from contracts, certification, conflicts, drift, owner, lineage, and documentation.
- AI readiness score: passed AI safety checks divided by total readiness checks.
